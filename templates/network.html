🌐 الفصل الثاني: الشبكات

🔌 ما هي الشبكة؟

شبكة = مجموعة أجهزة متصلة ببعضها لتبادل المعلومات.

💡 مثل بيوت الحارة… كل بيت فيه سكان، وكلهم متصلين بشبكة الماء أو الكهرباء. كذلك الأمر بالأجهزة!

📍 عنوان IP

هو الرقم يلي بيميز كل جهاز داخل الشبكة. مثل: 192.168.1.10

  • يشبه رقم هاتفك → يميزك داخل الشبكة
  • IP داخلي (داخل الشبكة) وIP خارجي (الإنترنت)

🆔 عنوان MAC

هو هوية الجهاز الثابتة، مثل بصمة الشبكة الخاصة بكرت الشبكة.

  • مثال: 00:1A:2B:3C:4D:5E
  • يُستخدم داخل الشبكة المحلية للتعرف على الأجهزة

🌍 ما هو DNS؟

DNS = Domain Name System. هو "دليل الهاتف" اللي بيحوّل أسماء المواقع إلى أرقام IP.

💡 لما تكتب google.com، المتصفح يسأل DNS: "شو الـ IP تبع هالموقع؟" فيرجعله مثلاً: 142.250.180.238

🔁 ما هو NAT؟

NAT = Network Address Translation

بيسمح لكل أجهزة بيتك تستخدم نفس الـ IP الخارجي، ويعرّفهم داخلياً بـ IPات محلية.

📦 الراوتر بيشتغل كموزّع ذكي: يوصل الإنترنت للّابتوب والموبايل والتلفزيون… كلهم يتشاركون نفس IP!

📶 LAN و WAN

  • LAN: شبكة محلية مثل شبكة بيتك أو المكتب
  • WAN: الشبكة العالمية – الإنترنت بأكمله

📚 نموذج OSI – 7 طبقات للاتصال

نموذج OSI يُقسّم الاتصال إلى 7 طبقات. كل طبقة لها وظيفة خاصة:

  1. التطبيق
  2. العرض
  3. الجلسة
  4. النقل
  5. الشبكة
  6. رابط البيانات
  7. الفيزيائية
✉️ تخيلها مثل رسالة بريدية: الفيزيائية = الشارع، الشبكة = صندوق البريد، التطبيق = محتوى الرسالة.

📡 بروتوكول TCP/IP

هو البروتوكول الحقيقي للإنترنت اليوم. يقسم البيانات إلى "رُزم" (Packets) وبيوصلها باستخدام:

  • TCP: يضمن ترتيب واستلام البيانات
  • IP: يحدد العنوان الرقمي للمستقبل
